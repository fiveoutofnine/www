## Icon button

To use icon-only buttons, use `IconButton`. It is identical to `Button`, except it does not have the `text` variant.

<DesignComponentsDisplay className="grid-flow-col grid-rows-2 md:grid-rows-1">
  <IconButton size="sm" children={<Circle />} />
  <IconButton size="md" children={<Circle />} />
  <IconButton size="lg" children={<Circle />} />
  <IconButton size="xl" children={<Circle />} />
</DesignComponentsDisplay>

## Link

To have the button act as a link, pass in a URL to the `href` prop. To open the link in a new tab, use the `newTab` prop. Note that the final component is rendered as a `<button>` element, not `<a>`.

<DesignComponentsDisplay className="grid-flow-col grid-rows-1">
  <Button children="Home" href="/" rightIcon={<ChevronRight />} />
  <Button children="New tab" href="/" rightIcon={<ExternalLink />} newTab />
</DesignComponentsDisplay>
