# Color

The design system uses [Radix UI colors](https://www.radix-ui.com/colors) to [ensure accessibility](https://www.radix-ui.com/colors/docs/palette-composition/composing-a-palette#choosing-text-scales) and [easy maintainability of consistent code/design](https://www.radix-ui.com/colors/docs/palette-composition/understanding-the-scale).

Additionally, the design system utilizes [`windy-radix-palette`](https://github.com/brattonross/windy-radix-palette), a Tailwind CSS plugin that adds Radix UI colors as Tailwind classes, to integrate with [Tailwind CSS](https://tailwindcss.com) easily.
